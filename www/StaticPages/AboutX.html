The Soil Moisture Information Processing System

The Soil Soil Moisture Processing System (SMIPS) make a number of modeled estimates of national soil water estimates freely available. It also makes a number of soil moisture probe data sets freely available.

The development of this system has been funded by the National Landcare program, along with CSIRO. All products are considered experimental and should be used with caution.

Soil Moisture Map Products

Currently there are 2 national soil moisture map products available in SMIPS
  1. The Soil Moisture Index
  2. The Volumetric Soil Moisture Estimate

The Soil Moisture Index

         Soil Moisture Integration and Prediction system (SMIPS) continuously generating 1-km root-zone moisture estimates for Australia in near real-time.

o   Integrates satellite data into a simple modelling framework

o   GPM IMERG 3B-HHR near-real-time satellite rainfall estimates statistically blended with BoM automatic weather station (AWS) to generate daily rainfall for the country at 1-km

o   Antecedent precipitation index (API) – basic AR(1) time series used to model surface soil layer moisture dynamics.

o   SMOS satellite top-layer soil moisture retrievals (L2SM) assimilated into API model.

o   API scaled between time series (1997-2012) min and max values to compute ‘wetness’ of top-layer

o   Exponential filter (recursive formulation) applied to wetness data to get  ‘root-zone wetness’ - - - note model parameters tuned to 0-90 cm average soil profile value
